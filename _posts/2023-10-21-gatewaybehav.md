# 对熵的粗略测量
均匀的随机数据将有接近相同数量的二进制的1比特和0比特，通过测量数据包内的比特的熵，可疑得到一个猜想：
- 1 bit 占 bit 数量 n%-N% 以下时，1bit 与 0 bit 刚好在敏感区间内，则触发审查

实际测量的的区间为 3.4 - 4.6：
![](/images/4ece0f5a-362d-4028-879b-179f6b87997d.png)

考虑到某些客户端在 TLS hello 中会有 0 填充的行为，每字节的 1 比特数通常较少，考虑谷歌浏览器105版发送的TLS Client Hello包，由于用零填充，每字节平均只有1.56个 1 比特，属于豁免范围

# 现实世界中的例子
- FTP，SSH 等强特征将被临时豁免。
- DNS-over-TCP由于包含很大一部分的 0 bit，处于豁免范围。
- `OPTIONS␣, DELETE␣, CONNECT␣, TRACE␣, PATCH` 均为可打印字符，后门跟着空格 0x20，请求方式的错误拼写（例如，TEG␣）不属于豁免范围。
